Коэффициент Фи (φ)

Коэффициент Фи (φ) измеряет силу связи между двумя бинарными переменными в таблицах 2×2. Является аналогом корреляции Пирсона для дихотомических данных.

Основные особенности

  • Только для таблиц 2×2
  • Диапазон значений: от -1 до +1
  • Показывает направление и силу связи
  • Связан с χ²: \( \phi = \sqrt{\frac{\chi^2}{n}} \)

Формула расчета

\[ \phi = \frac{ad - bc}{\sqrt{(a+b)(c+d)(a+c)(b+d)}} \]

Структура таблицы 2×2:

Событие+Событие-Всего
Группа 1aba+b
Группа 2cdc+d
Всегоa+cb+dn

Интерпретация:

  • φ = ±1: Полная связь
  • φ = 0: Нет связи
  • ±0.1: Слабая
  • ±0.3: Умеренная
  • ±0.5: Сильная

Пример расчета

Исследование связи курения с заболеваемостью:

БолеютЗдоровыВсего
Курят3070100
Не курят1090100
Всего40160200

Шаг 1. Идентификация переменных

\( a = 30,\ b = 70,\ c = 10,\ d = 90,\ n = 200 \)

Шаг 2. Подстановка в формулу

\[ \phi = \frac{(30×90) - (70×10)}{\sqrt{(30+70)(10+90)(30+10)(70+90)}} = \frac{2700 - 700}{\sqrt{100×100×40×160}} = \frac{2000}{16000} = 0.125 \]

Шаг 3. Интерпретация

φ = 0.125 указывает на слабую положительную связь между курением и заболеваемостью.

Важные замечания

  • Аналог корреляции для бинарных данных
  • Для таблиц больше 2×2 используйте V Крамера
  • Всегда проверяйте статистическую значимость
  • Не путать с коэффициентом Фи-квадрат

Ограничения метода

  • Неприменим для таблиц больше 2×2
  • Чувствителен к несбалансированным данным
  • Требует больших выборок для стабильности
  • Не показывает причинно-следственную связь
