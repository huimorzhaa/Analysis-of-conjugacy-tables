Коэффициент сопряженности Пирсона

Коэффициент сопряженности Пирсона (C) измеряет силу связи между двумя категориальными переменными. Принимает значения от 0 (полная независимость) до 1 (полная зависимость).

Основные особенности

  • Применим для таблиц любого размера
  • Основан на статистике χ²
  • Требует предварительного подтверждения значимости связи
  • Особенно полезен для таблиц 2×2

Формула расчета

\[ C = \sqrt{\frac{\chi^2}{\chi^2 + n}} \]

Где:

  • \( \chi^2 \) - значение хи-квадрат статистики
  • \( n \) - общее количество наблюдений

Интерпретация значений:

  • 0.00 - 0.10: Очень слабая связь
  • 0.10 - 0.30: Слабая связь
  • 0.30 - 0.50: Умеренная связь
  • 0.50 - 1.00: Сильная связь

Пример расчета

Исходные данные:

ДаНетВсего
Группа A301040
Группа B204060
Всего5050100

Шаг 1. Исходные параметры

\[ \chi^2 = 16.66,\quad n = 100 \]

Шаг 2. Подстановка в формулу

\[ C = \sqrt{\frac{16.66}{16.66 + 100}} = \sqrt{\frac{16.66}{116.66}} \approx \sqrt{0.1429} \approx 0.378 \]

Шаг 3. Интерпретация

Коэффициент \( C \approx 0.38 \) указывает на умеренную силу связи между переменными.

Сравнение с другими коэффициентами

Фи-коэффициент0-1Только для 2×2 таблиц

Важные примечания

  • Коэффициент всегда меньше 1 для таблиц с более чем 2 категориями
  • Не показывает направление связи - только силу
  • Рекомендуется использовать вместе с тестами на значимость
  • Для таблиц 2×2: \( C = \sqrt{\frac{\chi^2}{n + \chi^2}} \)
