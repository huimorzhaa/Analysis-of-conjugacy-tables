Изучение таблиц сопряженности методом хи-квадрат

Таблицы сопряженности (или таблицы связи) — это способ понять, связаны ли две категориальные переменные между собой. Например, зависят ли оценки студентов от того, на каком факультете они учатся.

Критерий хи-квадрат (\(\chi^2\))

Что важно учитывать:

  • Данные должны быть в виде частот: сколько раз что-то произошло, а не средние или проценты.
  • Независимость наблюдений: один человек — один ответ.
  • Ожидаемые частоты не должны быть слишком маленькими: желательно, чтобы в каждой ячейке таблицы было ожидаемо хотя бы 5 случаев.

Алгоритм расчета хи-квадрат по шагам:

  1. Построить таблицу сопряженности с фактическими частотами
  2. Вычислить ожидаемые частоты для каждой ячейки: \[ E_{ij} = \frac{(\text{итого по строке}) \times (\text{итого по столбцу})}{\text{общий итог}} \]
  3. Вычислить статистику хи-квадрат: \[ \chi^2 = \sum \frac{(O_{ij} - E_{ij})^2}{E_{ij}} \]
  4. Определить количество степеней свободы: \[ \text{степени свободы} = (r-1)(c-1) \] где \(r\) - число строк, \(c\) - число столбцов
  5. Сравнить с критическим значением из таблицы
  6. Сделать вывод:
    • Если \(\chi^2_{\text{расч}} > \chi^2_{\text{крит}}\) - связь есть
    • Если \(\chi^2_{\text{расч}} < \chi^2_{\text{крит}}\) - связи нет

Пример расчета

Исследуем связь между полом и выбором любимого напитка:

КофеЧайВсего
Мужчины301040
Женщины204060
Всего5050100

Ожидаемые частоты:

Для "Мужчины-Кофе": \[ E = \frac{40 \times 50}{100} = 20 \]

КофеЧайВсего
Мужчины202040
Женщины303060
Всего5050100

Расчет хи-квадрат:

\[ \chi^2 = \frac{(30-20)^2}{20} + \frac{(10-20)^2}{20} + \frac{(20-30)^2}{30} + \frac{(40-30)^2}{30} = 5 + 5 + 3.33 + 3.33 = 16.66 \]

Степени свободы:

\[ (2-1)(2-1) = 1 \]

Вывод:

Для 1 степени свободы и \(\alpha = 0.05\) критическое значение \(\chi^2 \approx 3.84\).

16.66 > 3.84 ⇒ связь между полом и выбором напитка статистически значима.
